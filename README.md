This is a game created by Tishya, Grace and Anerie during the JPEG Game Jam! 
You follow Kyle along a journey to make meme faces and scare away racoons.

Libraries:
- mediapipe
- open cv

Languages:
- html, js, css (frontend)
- python (backend)
- Flask API (full stack)
